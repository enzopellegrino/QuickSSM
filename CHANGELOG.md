# Changelog

All notable changes to QuickSSM will be documented in this file.

## [1.0.0] - 2025-12-05

### Added
- Multi-instance SSM session management
- Smart instance search and filtering
- Bulk instance selection with Select All/Deselect All
- AWS SSO profile discovery and management
- Profile deletion functionality
- Beautiful dark mode UI with QuickSSM branding
- Large centered logo when no terminals are open
- Alphabetical instance sorting
- Real-time terminal output with xterm.js
- Multiple terminal tabs support
- Automatic AWS CLI detection
- EC2 instance loading with visual spinner
- Error handling for expired AWS sessions
- macOS .icns icon support

### Features
- Connect to multiple EC2 instances simultaneously
- Quick profile switching
- Region selection
- Instance name display in tabs
- Terminal scrollback (2000 lines)
- Clean tab management with close buttons
- Responsive UI with flexbox layouts

### Technical
- Built with Electron 29
- AWS SDK v3 integration
- node-pty for terminal emulation
- xterm.js for terminal rendering
- Supports macOS 10.15+
